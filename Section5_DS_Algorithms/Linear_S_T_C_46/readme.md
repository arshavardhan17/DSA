## Linear Time – O(n)

The amount of **time** or **space** taken by the program **Increases or Decreases**, with your **input size increases or decreases**.

> This means the operation depend on the size of the input.

### ✅ Example:

```java
int getFirstElement(int[] arr) {
    return arr[0];
}
```
